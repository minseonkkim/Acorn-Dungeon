### 3.1.1 스프라이트 시트 표준 (전 캐릭터·적 공통)

모든 캐릭터(8명)와 적(잡몹·엘리트·보스 포함)은 **동일한 1행 × 24컷 가로 strip 레이아웃**을 따른다. AI 일러스트 생성 사이트의 표준 출력 사양(128×128 프레임)에 맞춰 시트는 **3072×128 PNG**로 받고, 게임에는 `AnimatedSprite2D` scale = 0.75로 **96×96 표시**한다.

#### 시트 레이아웃 (가로 24프레임 strip)

```
프레임 인덱스 0 ─────────── 23
[idle×4][walk×4][attack×4][hit×4][skill×4][death×4]
   0~3    4~7    8~11      12~15  16~19    20~23
```

각 프레임 128×128 → 시트 전체 폭 128 × 24 = 3072 px, 높이 128 px.

#### 시트 사양

| 대상                | 시트 사이즈 | 게임 내 표시 | scale | 비고                          |
| ------------------- | ----------- | ------------ | ----- | ----------------------------- |
| **캐릭터 8명**      | 3072×128    | 96×96        | 0.75  | 카툰 일러스트, nearest filter |
| **잡몹 12종**       | 3072×128    | 96×96        | 0.75  | 캐릭터와 동일 사양            |
| **엘리트 2종**      | 3072×128    | 128×128      | 1.0   | 원본 그대로, 색감·후광 차별화 |
| **보스(다람쥐 왕)** | 3072×128    | 192×192      | 1.5   | 페이즈별 시트 3장 별도        |

#### 프레임 인덱스 → 액션 매핑

| 인덱스 | 액션   | 사용 시점        | 비고                                |
| ------ | ------ | ---------------- | ----------------------------------- |
| 0~3    | idle   | 정지 시 루프     | 가장 자주 보임, 가장 신경써서       |
| 4~7    | walk   | 이동 중 루프     | 8fps × 4f = 0.5초 사이클            |
| 8~11   | attack | 자동 공격 발동   | 3번째 프레임에 발사체/히트박스 생성 |
| 12~15  | hit    | 피격 0.15초      | 1프레임만 사용해도 무방             |
| 16~19  | skill  | 액티브 스킬 발동 | 캐릭터별 무적 프레임 등 부여        |
| 20~23  | death  | 사망 1회 재생    | 마지막 프레임에서 정지 → 페이드     |

#### 좌우 방향

정면(혹은 측면) **1세트만 그리고 좌우는 `flip_h`로 반전**한다. 4방향 풀세트는 작업량 4배라 솔로 MVP 비-목표.

#### Godot 임포트 절차

1. AI 사이트에서 `3072×128, 24 frames horizontal strip` 사양으로 받은 PNG를 `assets/sprites/characters/<id>.png`에 저장
2. 임포트 옵션: **Filter Off, Mipmaps Off** (전역 `default_texture_filter=0`이라 자동)
3. 캐릭터별 `SpriteFrames` 리소스(`resources/sprite_frames/<id>.tres`)는 동일 템플릿 사용 — region 좌표 24개가 모두 `Rect2(128*n, 0, 128, 128)` 형태
4. `AnimatedSprite2D` 노드의 `scale`은 캐릭터·잡몹 0.75, 엘리트 1.0, 보스 1.5.

## 3.2 적(Enemy) 풀

### MVP 잡몹 12종 (던전 1 기준 6종 + 던전 1.5 보스용 6종)

| 등급   | 이름          | 속성 | 패턴                 | 메모           |
| ------ | ------------- | ---- | -------------------- | -------------- |
| 잡몹   | 도토리벌레    | 자연 | 직선 돌진            | 가장 약함      |
| 잡몹   | 도깨비풀      | 자연 | 정지 + 가시 발사     | 첫 원거리 적   |
| 잡몹   | 화난 다람쥐   | 자연 | 추적 + 회피          | 빠름           |
| 잡몹   | 솔방울괴물    | 대지 | 폭발 자폭            | 거리 유지 필요 |
| 엘리트 | 늙은 두꺼비   | 자연 | 점프 후 광역 충격파  | 보스급 디자인  |
| 엘리트 | 가시 고슴도치 | 대지 | 가시 회전 + 돌진     | 회피 중요      |
| 보스   | 다람쥐 왕     | 자연 | 3페이즈, 도토리 폭격 | 5층 보스       |

### 적 데이터 구조

```yaml
enemy:
  id: "spike_hedgehog"
  display_name: "가시 고슴도치"
  hp: 120
  atk: 8
  move_speed: 60
  element: "earth"
  weak_to: "fire"
  ai_pattern: "rolling_dash"
  drop_table:
    - { item: "acorn_leaf", weight: 60, qty: [1, 3] }
    - { item: "small_acorn", weight: 10, qty: [0, 1] }
  exp: 5
```

### AI 패턴 라이브러리

5가지로 한정. 모든 적은 이 중 하나 또는 조합.

| 패턴             | 설명                           |
| ---------------- | ------------------------------ |
| `straight_chase` | 플레이어 향해 직선 추적        |
| `keep_distance`  | 일정 거리 유지하며 원거리 공격 |
| `dash`           | 정지 → 차징 → 직선 돌진        |
| `area_attack`    | 일정 시간마다 자기 주변 광역   |
| `summon`         | 다른 잡몹 소환 (보스 전용)     |

> 솔로 개발에서는 AI를 단순하게. 5가지 + 조합으로 전 적 커버.

> 전체 룬 데이터는 `data/runes.json`에서 관리. 본 문서는 핵심 12개 + 진화 3개만 명시.

### 룬 디자인 가이드라인

- **스택 가능**: 단순 수치형은 3~5중첩 허용. 복잡한 효과는 1중첩.
- **명확한 시각화**: 룬 픽업 시 캐릭터 이펙트 변화 (불꽃 → 잔불 입자, 가시 → 외곽 가시).
- **빌드 파괴 카드**: 1런에 1번 정도 등장하는 "전 룬 리셋, 새 빌드" 카드. 매니아용.

## 3.4 아이템 / 재화

### 재화 종류

| 재화         | 획득처                 | 용도                              | 소프트/하드   |
| ------------ | ---------------------- | --------------------------------- | ------------- |
| 🌰 도토리    | 보스 클리어, 일부 미션 | 캐릭터·던전 언락, 메타 업그레이드 | 소프트 (희귀) |
| 🍃 도토리잎  | 잡몹·엘리트 처치       | 마을 상점, 룬 재추첨              | 소프트 (흔함) |
| 💎 별빛 가루 | IAP, 보상형 광고 일부  | 스킨, 부활, 추첨                  | 하드 (유료)   |

> 재화는 **3종으로 제한**. 라이트 유저 인지부담↓.

### 인게임 아이템 (런 중 일회성)

- **회복 도토리**: 즉시 HP 30% 회복
- **방패 잎**: 다음 1회 피격 무효
- **랜덤 룬 추첨권**: 다음 픽에 4장 중 1택 (보너스)

## 3.5 던전 콘텐츠

### MVP 출시: 던전 1 — 도토리 숲

- **층수**: 5
- **분위기**: 햇살이 비치는 평화로운 숲, 점점 어두워짐
- **음악**: 잔잔한 ukulele + 새소리
- **잡몹**: 도토리벌레, 도깨비풀, 화난 다람쥐, 솔방울괴물
- **엘리트**: 늙은 두꺼비, 가시 고슴도치
- **보스**: 다람쥐 왕 (3페이즈)

#### 보스 — 다람쥐 왕 (Boss: Acorn King)

```
페이즈 1 (HP 100~70%)
  - 도토리 던지기 (3연발)
  - 5초마다 박치기 돌진

  - 분노 모드. 이동속도+, 데미지+

보상:
  - 마법 도토리 1개 (확정)
  - 레어 룬 1개 (확정)
  - 별빛 가루 5개 (확률 30%)
```

## 3.6 업적 / 도감 시스템

라이트 유저의 장기 동기 부여.

### 도감 (Bestiary)

- 적·룬·캐릭터 처음 만나면 도감 등록
- 룬 진화 발견은 별도 코너 ("발견된 시너지 0/8")
- 100% 채우면 별빛 가루 100개 보너스
