## 한 줄 요약

> **"3분 던전, 따뜻한 동물들, 매번 다른 빌드"** — 출퇴근길 한 판 돌리고 캐릭터를 모으는 캐주얼 로그라이크.

## 핵심 디자인 원칙

1. **3~7분 한 런** — 라이트 유저의 출퇴근·자투리 시간에 맞춤.
2. **한 손 조작** — 가상 조이스틱 1개. 공격은 자동.
3. **매 런 다른 빌드** — 룬 카드 픽 + 랜덤 아이템으로 리플레이성 확보.
4. **귀여움이 핵심 IP** — 캐릭터 비주얼이 곧 마케팅이자 IAP 동기.
5. **솔로 개발 가능 스코프** — 시스템은 깊지만 콘텐츠는 작게, 데이터 주도 설계로 확장.
