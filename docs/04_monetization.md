# 04. 수익 모델 (Monetization)

## 4.1 모델 요약

> **무료 다운로드 + 보상형 광고 + 인앱결제(IAP)**

라이트 유저 친화적 하이브리드. 진입 장벽 0, 광고는 옵션, IAP은 편의·꾸미기에 집중.

### 매출 비중 목표 (출시 6개월 후)

| 채널 | 비중 | 비고 |
| --- | --- | --- |
| 보상형 동영상 광고 | 55% | 가장 큰 매출. eCPM 5~15달러 가정 |
| 전면 광고 | 10% | 던전 클리어 후 가끔 (페이스 조절) |
| IAP — 광고 제거 | 15% | $2.99 1회 결제. ARPU 견인 |
| IAP — 캐릭터·스킨 팩 | 15% | 패션 상품 |
| IAP — 별빛 가루 | 5% | 작게. P2W 방지 |

## 4.2 광고 설계

### 보상형 광고 (Rewarded Video) — 핵심

플레이어에게 **명확한 가치**를 주고 자발적 시청을 유도.

| 슬롯 | 보상 | 위치 | 일일 한도 |
| --- | --- | --- | --- |
| 부활 | 보스전 사망 시 1회 부활 (HP 50%) | 사망 화면 | 런당 1회 |
| 룬 재추첨 | 룬 카드 3장 다시 뽑기 | 룬 픽 화면 | 런당 2회 |
| 보상 2배 | 던전 클리어 보상 ×2 | 결과 화면 | 5회/일 |
| 데일리 미션 부스트 | 미션 보상 ×2 | 미션 화면 | 미션당 1회 |
| 무료 별빛 | 별빛 가루 +5 | 상점 | 3회/일 |

> **부활 광고가 가장 강력한 매출원**. 보스전 직전 사망의 분함 + 광고 시청의 가벼움 → 전환율 매우 높음.

### 전면 광고 (Interstitial) — 신중하게

- **트리거**: 3런마다 1회만, 10분에 1회 이상은 절대 X
- **타이밍**: 결과 화면 → 마을 진입 사이 (게임플레이 흐름 중간엔 절대 X)
- **광고 제거 IAP 가입자**: 노출 0
- **첫 30분**: 광고 0 (튜토리얼 보호)

### 배너 광고 — 사용 안 함

- 픽셀아트 화면 미관 손상, eCPM 낮음, 라이트 유저 이탈 요인. **MVP에 미적용**.

### 광고 SDK

- AdMob (기본). 미디에이션 없이 시작.
- 출시 후 6개월 시점에 ironSource 또는 Applovin MAX 미디에이션 검토.

## 4.3 인앱결제 (IAP) 설계

### 가격대 (KRW / USD)

| 상품 | 가격 | 종류 | 내용 |
| --- | --- | --- | --- |
| **광고 제거 팩** | ₩4,400 ($2.99) | 비소비성 | 모든 전면 광고 제거 + 별빛 50 |
| 별빛 가루 100 | ₩1,500 | 소비성 | (입문) |
| 별빛 가루 550 | ₩7,500 | 소비성 | 보너스 10% |
| 별빛 가루 1200 | ₩15,000 | 소비성 | 보너스 20% |
| 별빛 가루 3000 | ₩33,000 | 소비성 | 보너스 30% (최대) |
| **시작 팩** (1회만) | ₩3,300 | 비소비성 | 캐릭터 코코 + 스킨 1 + 별빛 100. 가성비 최고 |
| **곰 루루 캐릭터** | ₩5,500 | 비소비성 | 힐러 캐릭터 |
| 스킨 팩 1: 봄 | ₩3,300 | 비소비성 | 캐릭터 8명 봄옷 |
| **시즌 패스** (선택) | ₩9,900 | 기간성 30일 | 30단계 보상 트랙 |

### IAP 설계 원칙

1. **별빛 가루로 살 수 있는 것엔 P2W 요소 0**. 전투 능력은 IAP로 못 산다.
2. 별빛 가루는 **부활·재추첨·랜덤박스(스킨)**에만 사용. 영구 강화에는 사용 X.
3. **광고 제거 팩이 최고 가성비** 포지셔닝. 라이트 유저 한 번 결제 유도.
4. 시작 팩은 **첫 실행 후 24시간 한정**으로 노출, 1회만 구매 가능.

### 시즌 패스 (출시 +2개월 도입 검토)

- 30일 / 30단계
- 무료 트랙: 별빛 가루, 도토리잎
- 유료 트랙: 신규 스킨, 캐릭터 변형, 한정 룬 일러스트
- **전투 능력 보상은 0**. 패션과 자랑용만.

## 4.5 윤리 & 정책 가이드

전체이용가 등급 + 라이트 유저 타겟 → **공격적 BM 금지**.

- ❌ 가챠(랜덤박스) 자체 미도입. 스킨은 직접 구매만.
- ❌ 시간 압박형 팝업 ("3분 안에 사면 50% 할인!")
- ❌ 푸시 알림 BM 유도 ("당신의 캐릭터가 외로워해요!")
- ❌ "에너지" 시스템 (대기시간으로 결제 강요)
- ✅ 광고는 항상 **자발적 + 보상 명시**
- ✅ 12세 이하에 대한 IAP은 OS 레벨 정책 의존, 자체 결제벽 없음
